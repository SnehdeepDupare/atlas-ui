## December 2025 - CLI support and enhancements

### Framework Upgrade

All React components and the website have been upgraded to **Next.js 16** and **React 19**.

### CLI Support

You can now use the **shadcn CLI** to add Atlas UI components directly to your project. The CLI supports both **React** and **Vanilla JavaScript** components.

**Example:**

```bash
npx shadcn@latest add https://atlasui.vercel.app/r/reveal-effect.json
```

Explore more components at [here](https://atlasui.vercel.app/docs/components).

### v0 Integration

All react components are editable on [v0](https://v0.app). This allows you to easily customize the components in natural language and paste into your app.
