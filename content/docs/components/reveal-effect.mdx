```js
import { animate, inView } from "motion";

const speedDurations = {
  slow: 1.6,
  medium: 1,
  fast: 0.5,
};

document.querySelectorAll("#reveal-effect").forEach((el) => {
  const delay = parseFloat(el.dataset.delay) || 0;
  const speed = el.dataset.speed || "medium";
  const translateY = parseFloat(el.dataset.translatey) || 0;

  el.style.opacity = "0";
  el.style.filter = "blur(12px)";
  el.style.clipPath = "inset(-5% 100% -5% 0)";
  el.style.transform = `translateY(${translateY}px)`;

  inView(el, () => {
    animate(
      el,
      {
        opacity: 1,
        filter: "blur(0px)",
        clipPath: "inset(-5% 0% -5% 0)",
        transform: "translateY(0)",
      },
      {
        duration: speedDurations[speed],
        easing: "ease-in-out",
        delay,
      }
    );
  });
});
```

## Usage

<UsageTabs>

<TabsContent value="React/Next">

```tsx
import { RevealEffect } from "@/components/reveal-effect";
```

```tsx
<RevealEffect>
  <h1>Reveal Effect</h1>
</RevealEffect>

<RevealEffect>
  // Pass any children here, and the reveal effect will be applied to it.
</RevealEffect>
```

You can increase/decrease the speed of the animation by passing the `speed` prop.

</TabsContent>

<TabsContent value="JavaScript">

Assign the `id="reveal-effect"` to any element to apply the reveal effect.

```html
<div id="reveal-effect">
  <h1>Reveal Effect</h1>
</div>
```

</TabsContent>

</UsageTabs>

## Props / Data Attributes

<UsageTabs>

<TabsContent value="React/Next">

| Prop         | Type              | Default    | Description                                                                  |
| ------------ | ----------------- | ---------- | ---------------------------------------------------------------------------- |
| `children`   | `React.ReactNode` | -          | The content to which animation must be applied                               |
| `speed`      | `string`          | `"medium"` | Animation speed, can be `"slow"`, `"medium"`, or `"fast"`                    |
| `delay`      | `number`          | `0`        | Delay before the animation starts                                            |
| `translateY` | `number`          | `0`        | The Y-axis translation from where the element will initially begin animation |

</TabsContent>

<TabsContent value="JavaScript">

| Data Attribute    | Type     | Default    | Description                                                                  |
| ----------------- | -------- | ---------- | ---------------------------------------------------------------------------- |
| `data-speed`      | `string` | `"medium"` | Animation speed, can be `"slow"`, `"medium"`, or `"fast"`                    |
| `data-delay`      | `number` | `0`        | Delay before the animation starts                                            |
| `data-translatey` | `number` | `0`        | The Y-axis translation from where the element will initially begin animation |

</TabsContent>

</UsageTabs>

## Example

### Speed

To change the speed of the animation, you can pass the `speed` prop or use the `data-speed` attribute.
The default speed is `"medium"`. You can set it to `"slow"` or `"fast"`.

### Translate Y

To change the Y-axis translation from where the element will initially begin animation, you can pass the `translateY` prop or use the `data-translatey` attribute.
The default value is `0`. You can set the value between `10-15` for a subtle effect or any other value suitable for your needs.
Pair `translateY` with a small `delay` for a more pronounced effect.
