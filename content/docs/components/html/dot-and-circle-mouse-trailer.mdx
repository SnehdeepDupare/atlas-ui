## Installation

<InstallationTabs>

<TabsContent value="cli">

```bash
npx shadcn@latest add https://atlasui.dev/r/dot-and-circle-mouse-trailer-html.json
```

</TabsContent>

<TabsContent value="manual">

<Steps>

<Step>Copy and paste the following code into your project.</Step>

```html
<div class="trailer" id="trailer"></div>
<div class="trailer-outline" id="trailer-outline"></div>
```

<Step>Add `style.css`</Step>

```css
body {
  margin: 0;
  padding: 0;
  height: 100vh;
  width: 100%;
  background-color: #18181b;
}

.trailer {
  position: fixed;
  width: 8px;
  height: 8px;
  background-color: white;
  border-radius: 50%;
  left: 0;
  top: 0;
  transform: translate(-50%, -50%);
  pointer-events: none;
  z-index: 1000;
}

.trailer-outline {
  position: fixed;
  left: 0;
  top: 0;
  transform: translate(-50%, -50%);
  width: 80px;
  height: 80px;
  border: 2px solid white;
  border-radius: 50%;
  pointer-events: none;
  z-index: 1000;
}
```

<Step>Add `script.js`</Step>

```js
const trailer = document.getElementById("trailer");
const trailerOutline = document.getElementById("trailer-outline");

document.body.onpointermove = (e) => {
  const { clientX, clientY } = e;

  trailer.animate(
    {
      left: `${clientX}px`,
      top: `${clientY}px`,
    },
    {
      duration: 1000,
      fill: "forwards",
    }
  );

  trailerOutline.animate(
    {
      left: `${clientX}px`,
      top: `${clientY}px`,
    },
    {
      duration: 3000,
      fill: "forwards",
    }
  );
};
```

</Steps>

</TabsContent>

</InstallationTabs>

## Usage

Place the following code in your `index.html` to use the Mouse Trailer.

```html
<div class="trailer" id="trailer"></div>
<div class="trailer-outline" id="trailer-outline"></div>
```
