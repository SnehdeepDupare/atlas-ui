## Usage

Wrap any element with a `<div id="box-reveal">` to apply the box reveal effect.

```html
<div id="box-reveal">
  <h1>Box Reveal</h1>
</div>
```

## Data Attributes

| Data Attribute | Type                      | Default         | Description                                     |
| -------------- | ------------------------- | --------------- | ----------------------------------------------- |
| `data-side`    | `"left" \| "right"`       | `"left"`        | Direction from which the reveal box slides away |
| `data-color`   | `string`                  | `"#FACC15"`     | Background color of the reveal overlay          |
| `data-width`   | `"fit-content" \| "full"` | `"fit-content"` | Controls the width of the reveal container      |
