## Usage

```tsx
import { CascadeText } from "@/components/cascade-text";
```

```tsx
<CascadeText>
  <h1>Your text here</h1>
</CascadeText>

<CascadeText>
  <p>Your description here</p>
</CascadeText>

<CascadeText>
  {/* Pass any text element */}
</CascadeText>
```

## Props

| Prop              | Type              | Default | Description                                                                                                                                                                                   |
| ----------------- | ----------------- | ------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `children`        | `React.ReactNode` | -       | The text content to animate                                                                                                                                                                   |
| `animateOnScroll` | `boolean`         | `true`  | Controls when the animation starts. When `true`, the animation triggers as the element enters the viewport (around the lower 75% of the screen). When `false`, it plays immediately on mount. |
| `delay`           | `number`          | `0`     | Delay before the animation starts                                                                                                                                                             |
