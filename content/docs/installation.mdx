## Add Utilities for React, Next.js, etc. projects

<Steps>

<Step>Install dependencies</Step>

```bash
npm i motion clsx tailwind-merge
```

<Step>Add util file</Step>

`lib/utils.ts`

```ts
import { ClassValue, clsx } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}
```

<Step>You are good to go and start copy pasting the components into your project.</Step>
</Steps>

## Vanilla JavaScript

<Steps>
<Step>Add the following script to your project head section to use Motion library.</Step>

```html
<script src="https://cdn.jsdelivr.net/npm/motion@latest/dist/motion.js"></script>
```

<Step>
  You are good to go and start copy pasting the components into your project.
</Step>

</Steps>
